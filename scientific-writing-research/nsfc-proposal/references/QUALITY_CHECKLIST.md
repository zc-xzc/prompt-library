# NSFC 申请书 · 提交前质量自检清单

按 2026 年度最新版 NSFC 规范整理，Agent 在交付草稿前应主导用户逐项核对。

---

## 一、形式完整性（不予受理风险项）

- [ ] 申请代码已选至**二级**（4位数字），且为生命科学部等强制要求学部所允许
- [ ] 资助类别、亚类（C/B/A）、附注说明填写一致
- [ ] **正文不超过 30 页**
- [ ] 摘要 ≤ 400 字
- [ ] 关键词 3-5 个，覆盖研究对象 / 核心方法 / 关键科学问题
- [ ] 研究期限：C/B类 = 3 年；A类 = 5 年
- [ ] 主要参与者已**在线填写简历并上传 PDF**
- [ ] 5 篇代表性论文已上传**公开发表全文 PDF**
- [ ] 涉及人体组织/细胞/器官 → 已上传伦理委员会审核证明
- [ ] 涉及动物实验 → 部分科学部已提供伦理证明
- [ ] 涉及高致病性病原体 → 已上传生物安全保障承诺书
- [ ] 涉及人类遗传资源 → 已提供《人类遗传资源管理条例》相关说明
- [ ] A类临床科学申请人 → 已上传医师资格证、执业证、高级职称聘任证
- [ ] 申请人 + 主要参与者均无超项

---

## 二、内容质量

### 2.1 摘要

- [ ] 陈述式语气，未使用"拟""将""可能"等不确定表述
- [ ] 涵盖背景 → 不足 → 内容 → 方法 → 预期成果五要素
- [ ] 信息密度足够，无套话

### 2.2 立项依据

- [ ] **开篇 500 字内**清晰呈现：原创性 / 与现有研究的本质区别 / 科学价值
- [ ] 文献综述**有评有述**（"评"多于"述"），每段揭示研究缺口
- [ ] 国内外现状分别成段，覆盖近 5 年核心文献
- [ ] 提出的核心科学问题属于**基础研究层面**的 "why" / "how"，而非工程 "how to do"
- [ ] C 类：体现申请人独立提出问题的能力，不靠成果堆积"撑场面"
- [ ] B/A 类：有充分前期工作支撑，体现系统性与深度

### 2.3 研究内容

- [ ] 所有子内容统一于核心科学问题，呈逻辑递进而非平行罗列
- [ ] **不预设固化提纲**（2026 改革要求）
- [ ] 研究目标具体、可量化或可评估，3 年内可实现
- [ ] 关键研究方法已**融入**研究内容叙述（取代独立"研究方案"板块）
- [ ] 关键技术路线图保留 1-2 张
- [ ] 创新点 2-4 条，每条含 [类型 / 现状 / 突破 / 意义] 四要素
- [ ] 每条创新点均有**文献对比支撑**，无"国内首次""填补空白"等空话
- [ ] 年度计划保留（简表即可）

### 2.4 研究基础

- [ ] 与本项目直接相关的初步实验数据 / 预研结果已展示
- [ ] **5 篇代表作**：≥3 篇二区及以上，避开预警 / On-hold 期刊
- [ ] C 类：≥2 篇唯一一作的二区以上论文
- [ ] 每篇代表作均有**本人贡献声明**（具体内容 + 工作量百分比）
- [ ] 5 篇形成"机制发现 → 验证 → 拓展"完整研究链条
- [ ] 在研项目情况：明确说明与本申请的**区别**，避免重复资助嫌疑
- [ ] 工作条件描述紧扣研究内容，不罗列所有设备

### 2.5 个人简历（在线填写部分）

- [ ] 职称按时间段详细列出，无跳跃
- [ ] 学位时间以证书日期为准
- [ ] 博士后人员职称选"无"
- [ ] 论文作者顺序、署名情况（唯一/共同）真实标注
- [ ] 近五年项目：资助起始日期晚于 2021-01-01 或目前在研

---

## 三、诚信合规

- [ ] 所有信息真实、准确，与证件 / 证书一致
- [ ] 未使用 AI 直接生成申请书内容
- [ ] 如使用 AI 辅助（文献检索 / 整理），已**全面如实声明**并按规定标识
- [ ] 全文无涉密、涉政敏感内容
- [ ] 严禁虚假标注第一作者 / 通讯作者
- [ ] 严禁篡改作者顺序

---

## 四、A 类（杰青）附加项

- [ ] 每年在依托单位从事研究工作时间 ≥ 9 个月
- [ ] 延续资助申请代码与第一资助周期**同一科学部**
- [ ] 延续资助附第一周期结题报告与代表性成果

---

## 五、Agent 交付前的最后两问

完成上述清单后，Agent 应主动向用户确认：

1. "申请书 PDF 是否已经过申请人本人逐字校对？AI 生成的措辞与事实是否全部经过核实？"
2. "依托单位科研处的形式审查时间窗是否已对接？所有附件 PDF 文件命名是否符合单位要求？"

若任一项为"否"，请提示用户先解决合规问题再提交。
