# 国社科申请书 · 提交前质量自检清单

按 2025 年度全国哲学社会科学工作办公室官方模板整理。Agent 在交付草稿前应主导用户逐项核对。

---

## 一、形式完整性

- [ ] **课题名称 ≤ 40 字**，无副标题，直接反映核心研究问题
- [ ] 项目类别（A/B/C/X）填写正确
- [ ] 学科分类已选至**二级学科**（含代码 + 名称）
- [ ] 研究类型（基础 / 应用 / 综合）与课题性质一致
- [ ] 关键词 ≤ 3 个，空格分隔
- [ ] 申请经费：A 类 35 万；B/C/X 类 20 万
- [ ] 计划完成时间：基础研究 3-5 年；应用研究 2-3 年
- [ ] 预期成果形式填写（专著 / 论文集 / 论文 / 研究报告）
- [ ] 申请人资格符合所申报类别要求（青年项目须 35 岁以下）
- [ ] 申请人 + 课题组无超项

---

## 二、内容质量

### 2.1 选题说明（2025 年新增模块，评审首读内容）

- [ ] **总字数 ≤ 300 字**
- [ ] 一段话连贯表达，**未分条**
- [ ] 涵盖三要素：①核心研究问题 ②研究视角/理论框架 ③核心概念界定
- [ ] 高度凝练，直击要害
- [ ] **不含**作者姓名、单位名称等可识别身份信息

### 2.2 选题依据（约 1500-2000 字）

- [ ] **学术史梳理**：
  - [ ] 国际研究分 2-3 派 / 阶段，含代表学者与代表性观点
  - [ ] 国内研究分 2-3 维度，评述成就与不足
  - [ ] 已有国社科同类项目（近 5 年）已梳理，并指出本研究差异
- [ ] **"述：评 = 3:7"**，每段末有揭示研究缺口的转折句
- [ ] 引用文献格式规范（GB/T 7714 或国际通用格式，全文统一）
- [ ] **学术价值**列 2-3 点：填补空白 / 拓展理论 / 提供新框架
- [ ] **应用价值**列 1-2 点：政策依据 / 决策参考 / 回应国家战略

### 2.3 研究内容（约 1500 字）

- [ ] **主要研究目标** 3 个（理论 / 实证 / 政策）
- [ ] 已出现**提纲或目录**（部分 / 章节框架可视化）
- [ ] **重点与难点**已说明，含具体解决方案
- [ ] **研究计划**按年度展开（表格形式），与"计划完成时间"严格一致
- [ ] **可行性论证**覆盖：理论积累 / 数据基础 / 团队配置 / 时间保障

### 2.4 创新之处（约 600 字）

- [ ] 分条列举，每条有实质内容
- [ ] 从三维度各选 1-2 点：
  - [ ] 学术观点创新（新论断 / 新命题）
  - [ ] 研究方法创新（方法组合 / 新框架）
  - [ ] 研究视角或议题创新（交叉视角 / 新兴议题）
- [ ] 无"国内首次""填补空白"等空洞表述
- [ ] 每条创新点与正文内容呼应

### 2.5 预期成果（约 200 字）

- [ ] 列明成果形式、数量、字数
- [ ] 活页版**未填具体出版社名 / 期刊名**

### 2.6 参考文献（10-20 条）

- [ ] 格式规范（GB/T 7714）
- [ ] 与正文引用严格对应，无堆砌
- [ ] 近 5 年文献占比合理

### 2.7 研究基础

- [ ] **学术简历**（约 400 字）：仅写与本课题相关的学术经历
- [ ] **前期成果限 5 项**：与本课题高度相关
- [ ] 每项前期成果均说明"与本课题的关联"
- [ ] **承担项目情况**：表格形式列出
- [ ] **与已有项目 / 博士论文的关系**：如有，明确"实质性拓展"，文字重复率 < 60%；如无，注明"无"

### 2.8 经费概算

- [ ] 总额：A 类 35 万；B/C/X 类 20 万
- [ ] **间接费用** ≤ 总经费 40%
- [ ] **劳务费** ≤ 直接费用 30%
- [ ] 设备费已压缩
- [ ] 每个开支科目均有**用途说明**
- [ ] 经费合理，无虚报

---

## 三、活页版匿名化（如需提交活页）

- [ ] **总字数 ≤ 7000 字**（含选题说明）
- [ ] **申请人身份信息**已删除：
  - [ ] 申请人姓名
  - [ ] 所在单位名称
  - [ ] 所在城市（用"某高校""某省"代替）
- [ ] **前期成果**仅列：
  - [ ] 成果名称
  - [ ] 与本课题的关联说明
  - [ ] **未列**发表刊物名、出版社名、合作者姓名
- [ ] **自引文献**已改为匿名表述（"已有研究"等）
- [ ] 全文检索"我""本人主持"等可推断身份的表述，已清除或改写
- [ ] 课题组成员列表已删除或改为"课题组成员 N 人"

---

## 四、诚信合规

- [ ] 所有信息真实、准确
- [ ] **未使用 AI 直接生成申请书内容**
- [ ] 如使用 AI 辅助（文献检索 / 整理），已**全面如实声明**
- [ ] 申请人 + 课题组成员均无**超项**情况
- [ ] 与在研项目 / 已结项目无重复资助嫌疑
- [ ] 全文无涉密、涉政敏感内容

---

## 五、Agent 交付前的最后三问

完成上述清单后，Agent 应主动向用户确认：

1. "选题说明（300字）是否经过反复打磨？这是评审专家首先阅读的内容，质量直接影响第一印象。"
2. "活页版是否经过逐字筛查身份信息？特别是自引文献和'我'字开头的句子。"
3. "依托单位科研处的形式审查时间窗是否已对接？申报系统的提交截止日期是否已确认？"

若任一项为"否"，请提示用户先解决合规问题再提交。

---

## 六、不同项目类别的差异化提醒

### 重点项目（A）

- [ ] 选题须紧扣"重大理论与现实问题"
- [ ] 研究团队配置须体现学科代表性
- [ ] 经费 35 万元的使用规划须有匹配的工作量

### 一般项目（B）

- [ ] 选题须有学科前沿性与创新性
- [ ] 研究方法须严谨、可执行

### 青年项目（C）

- [ ] 申请人年龄符合（35 岁以下）
- [ ] 体现申请人的学术训练与创新潜力
- [ ] 不必过度强调资历，重在"问题意识"

### 西部项目（X）

- [ ] 申请人为西部省区在职人员
- [ ] 选题与西部经济社会发展直接相关
- [ ] 体现对地方实际问题的回应
