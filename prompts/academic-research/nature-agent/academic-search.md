# 文献检索与综述

用于制定检索式、整理候选文献、扩展引文网络和形成可复核的综述框架。

## 可直接使用的提示词

```text
请为以下研究问题设计一份可复核的文献检索与综述方案：<研究问题>。

先给出数据库或权威来源、检索式、时间范围、语言范围、纳入与排除标准；随后用表格列出每篇候选文献的题名、作者、年份、来源、DOI 或稳定链接、与问题的相关性和核验状态。不要把未经确认的条目写成真实文献。

在形成综述时，按研究问题、方法、主要发现、局限和相互矛盾之处分组；区分原始研究、综述、预印本和二手转述。最后列出需要我人工打开原文核对的条目与原因。
```

## 核验要求

- 优先使用出版商页面、DOI 注册信息、数据库记录和论文原文。
- 引文扩展只能作为发现候选资料的方法，不能替代阅读和核验。
- 不可访问全文时，明确只依据了摘要或索引信息。
