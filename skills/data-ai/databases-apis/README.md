# 🔗 Databases & APIs（数据库与 API）

| 技能 | 说明 |
|------|------|
| [alpha-vantage](alpha-vantage/) | Alpha Vantage 实时与历史金融数据 |

| [fred-economic-data](fred-economic-data/) | FRED 美联储经济数据 |
| [get-available-resources](get-available-resources/) | 获取可用资源列表 |
| [hedgefundmonitor](hedgefundmonitor/) | OFR 对冲基金监控数据 |
| [openalex-database](openalex-database/) | OpenAlex 学术文献数据库 |
| [parallel-web](parallel-web/) | 并行网络搜索 |
| [perplexity-search](perplexity-search/) | AI 驱动的网络搜索 |
| [pubmed-database](pubmed-database/) | PubMed 生物医学文献 |
| [usfiscaldata](usfiscaldata/) | 美国财政数据 |
| [uspto-database](uspto-database/) | USPTO 专利数据库 |
