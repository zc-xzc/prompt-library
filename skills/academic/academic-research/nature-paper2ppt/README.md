# Nature Paper2PPT

将科学论文自动转化为中文 PPTX 演示文稿，采用 Nature 风格的报告逻辑。支持期刊俱乐部、组会、论文分享、学术报告等场景。

## 使用场景

- 文献汇报 / Journal Club
- 组会论文分享
- 毕业论文答辩
- 学术会议报告
- 研究方法论介绍

## 核心特点

- 论文科学论证线作为 PPT 逻辑主线
- 中文学术幻灯片 + 讲者备注
- 提取并嵌入论文原图
- 12-16 页标准结构
- 跨平台 Python 工具链
