# Nature Citation

为学术论文文本添加严格 Nature/CNS 系列引文。将长段落拆分为可引用片段，检索 Nature Portfolio、AAAS Science 系列和 Cell Press 期刊，输出参考文献管理文件。

## 使用场景

- 论文写作为段落寻找 CNS 系列支撑文献
- 分段引用验证和补引用
- 导出 EndNote / RIS / Zotero 格式引用文件
- Nature / Science / Cell 及子刊范围检索
