# Nature Academic Search

多源学术文献检索框架。通过 MCP 工具（PubMed、CrossRef、arXiv、Semantic Scholar 等）进行协调式多步骤文献检索、引用验证、MeSH 搜索策略制定和引用文件管理。

## 使用场景

- 多源文献检索（PubMed + CrossRef + arXiv 联合搜索）
- 引用验证和溯源
- MeSH 搜索策略构建
- 引用格式转换（.nbib/.ris/.bib）
- BibTeX 和参考文献管理
