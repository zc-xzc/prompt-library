# 学术论文写作

用于按论文结构起草内容，但不把生成文本当作已验证的研究结论。

## 可直接使用的提示词

```text
请协助起草论文的 <摘要 / 引言 / 方法 / 结果 / 讨论> 部分。

研究主题：<填写>
目标读者或期刊：<填写>
已有的已核验材料：<粘贴数据、方法、结论或文献要点>
长度与格式要求：<填写>

先列出该部分应回答的问题和建议结构；只依据我提供或已核验的材料写作。对证据不足的地方使用清晰占位标记，例如 [需要补充来源] 或 [需要核对数据]，不要补造事实或引用。语言应具体、克制，避免将相关性写成因果性，避免夸大新颖性或实际意义。最后附上事实核验清单。
```

## 写作检查

- 区分研究结果、解释、假设和局限。
- 每个引用应能回溯到真实、可访问或明确标注为待核验的来源。
- 方法、样本、统计和伦理信息不得由模型臆测补全。
