# Nature Agent：学术研究工作流

## 可直接使用的提示词

```text
请以可核验的学术研究工作流协助我完成任务。先判断任务属于：文献检索、综述梳理、论文写作、文本润色、引文管理，或它们的组合；再把工作拆成可检查的阶段。

无论任务类型，都必须遵守：
1. 将已核验的证据、暂未核验的信息、推测和待补充内容明确分开；
2. 不编造论文、DOI、作者、实验结果、数据、引文或参考文献；
3. 任何检索结果、引文和事实主张在用于论文、投稿或公开材料前，逐条以权威原始来源核验；
4. 当资料不足时，说明缺口并给出下一步检索或补证方案；
5. 在结束时提供阶段结果、未核验项和下一步清单。

我的任务：<填写研究问题、目标期刊或课程要求、已有资料、截止时间和期望交付物>
```

## 工作流路由

- 需要找文献、比较研究脉络或做综述：使用 [文献检索与综述](academic-search.md)。
- 需要起草摘要、引言、方法、结果或讨论：使用 [学术论文写作](academic-writing.md)。
- 需要检查语言风格、过度主张或句子可读性：使用 [Nature 风格润色](academic-polishing.md)。
- 需要整理参考文献、统一格式或检查引文位置：使用 [引文管理与引用句](academic-citation.md)。

## 使用边界

这不是自动投稿工具，也不替代研究者的文献核验、研究设计、统计判断、署名责任或学术伦理审查。若来源不可访问、版权受限或信息互相矛盾，应如实说明，而不是补写看似合理的内容。
