# 引文管理与引用句

用于统一参考文献格式、检查引文与论断的对应关系，并起草需要人工核验的引用句。

## 可直接使用的提示词

```text
请协助我整理以下文献与正文引用。目标格式：<Nature / APA / Vancouver / IEEE / GB/T 7714 / 其他>。

先为每条文献保留题名、作者、年份、期刊或出版社、卷期页、DOI 或稳定链接及核验状态；再检查每个正文论断是否有对应来源，以及来源是否真的支持该论断。缺少或无法核验的信息请明确标记，不要猜测 DOI、页码、作者顺序或出版信息。

如需生成引用句，请提供中性、可追溯的表述，并将其标为“待作者核验”。最后输出：格式化参考文献、正文引用建议、待核验清单和发现的错配项。

文献与正文：<粘贴>
```

## 使用边界

格式正确不代表内容真实。引用管理工具和模型输出均可能出错；投稿前应以原文、期刊要求和正式文献管理器导出的记录为准。
