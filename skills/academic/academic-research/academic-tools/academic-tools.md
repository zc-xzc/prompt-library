# Academic Tools：研究工具选择与使用

## 可直接使用的提示词

```text
请帮我为下面的学术资料处理任务制定一个安全、可验证的本地工具使用方案。

任务：<例如政策文本统计、文件批量重命名、PDF 整理、表格合并或文本预处理>
输入位置与格式：<填写>
期望输出：<填写>
是否允许覆盖原文件：否，除非我随后明确确认
可用工具或仓库：https://github.com/zc-xzc/academic-tools

请先只做规划与检查：识别适合的工具模块、依赖、权限与版权风险；列出精确输入范围、预计改动、输出位置和备份方案。对于批量重命名、合并、拆分、自动点击或下载操作，必须先生成预览或小样本结果，等待我确认后才执行。

不得绕过网站访问限制、验证码、付费墙或版权限制；不得把未经授权下载的资料当作可以自由使用的内容。结束时给出验证方法、未处理项和可回滚方式。
```

## 适用范围

可用于筛选或规划政策文本分析、文件改名与合并、PDF 整理、表格处理等本地任务。实际代码、依赖和第三方材料不在本仓库维护；请到 [源仓库](https://github.com/zc-xzc/academic-tools) 获取并审阅。
