# Academic Tools：研究工具选择与使用

这是一个研究自动化与文本处理工具的使用入口，对应独立项目 [`zc-xzc/academic-tools`](https://github.com/zc-xzc/academic-tools)。原项目包含 Python 脚本、政策文本分析、文件处理和下载辅助组件；本仓库只提供安全的任务规划提示词，不复制其源代码或第三方材料。

完整工具、安装说明、许可证和第三方告知应始终以 [源仓库](https://github.com/zc-xzc/academic-tools) 及其 [第三方说明](https://github.com/zc-xzc/academic-tools/blob/main/THIRD_PARTY_NOTICES.md) 为准。

## 使用前注意

- 先在副本或小样本上测试，确认输入、输出目录和覆盖策略。
- 不要用自动化绕过数据库、网站或内容提供方的访问限制；下载或处理受版权保护内容前确认合法授权。
- 批量重命名、合并、拆分和下载都可能影响大量文件，执行前应列出精确目标并保留可恢复备份。
