# 🧬 Life Sciences（生命科学）

生物信息学、化学信息学、药物发现和临床医学。

## 分类

| 分类 | 数量 | 说明 |
|------|------|------|
| [bioinformatics](bioinformatics/) | 68 技能 | 基因组学、蛋白质组学、单细胞、系统生物学等 9 个子领域 |
| [cheminformatics-drug-discovery](cheminformatics-drug-discovery/) | 14 技能 | 分子处理、药物设计、化学数据库 |
| [clinical-medical](clinical-medical/) | 6 技能 | 临床决策、报告撰写、治疗方案、法规认证 |

## 快速使用

```bash
# 安装生物信息学技能
cp -r life-sciences/bioinformatics/genomics/scanpy ~/.codex/skills/

# 安装化学信息学技能
cp -r life-sciences/cheminformatics-drug-discovery/rdkit ~/.codex/skills/
```
