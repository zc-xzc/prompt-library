# 🧬 Bioinformatics（生物信息学）

最大的技能分类，包含 68 个生物信息学相关技能，分为 9 个子领域。

## 子分类

| 子目录 | 技能数 | 涵盖内容 |
|--------|--------|----------|
| [genomics](genomics/) | 14 | Arboreto, ClinVar, DeepTools, ENA, Ensembl, Gene, Geniml, GEO, gnomAD, GTars, GTEx, GWAS, JASPAR, PySAM, TileDB-VCF |
| [proteomics](proteomics/) | 7 | AlphaFold-DB, ESM, InterPro, PDB, PyOpenMS, STRING, UniProt |
| [single-cell](single-cell/) | 7 | AnnData, CELLxGENE, FlowIO, HistoLab, PyDESeq2, Scanpy, scVelo, scvi-tools |
| [systems-biology](systems-biology/) | 5 | BioServices, BRE nda, COBRApy, KEGG, Reactome |
| [bio-databases](bio-databases/) | 10 | bioRxiv, cBioPortal, ClinPGx, COSMIC, DepMap, HMDB, IDC, Metabolomics, Monarch, OpenTargets |
| [lab-automation](lab-automation/) | 10 | Adaptyv, Benchling, DNAnexus, LabArchive, LaminDB, LatchBio, OMERO, Opentrons, Protocols.io, PyLabRobot |
| [neuro-imaging](neuro-imaging/) | 4 | NeuroKit2, Neuropixels, PathML, PyDICOM |
| [phylo-evo](phylo-evo/) | 3 | ETE Toolkit, Phylogenetics, Scikit-Bio |
| [bio-utils](bio-utils/) | 5 | BGPT, gget, GlycoEngineering, PyHealth, Scikit-Survival |

## 快速使用

```bash
# 安装单个技能
cp -r genomics/scanpy ~/.codex/skills/

# 或在 Codex 中直接对话
"请安装 scanpy 技能用于单细胞数据分析"
```
