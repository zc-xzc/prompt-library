# Nature Agent：学术研究工作流

这是一个面向学术研究任务的工作流索引，覆盖文献检索、论文写作、文本润色和引文管理。它来自独立项目 [`zc-xzc/nature-agent`](https://github.com/zc-xzc/nature-agent)，本仓库仅归档可复用的技能说明，不包含其 Python 实现。

## 快速使用

从 [主工作流](nature-agent.md) 开始，根据任务打开对应专项说明：

1. [文献检索与综述](academic-search.md)
2. [学术论文写作](academic-writing.md)
3. [Nature 风格润色](academic-polishing.md)
4. [引文管理与引用句](academic-citation.md)

## 重要限制

原项目当前是实验性原型：默认检索、DOI 查询和生成内容可能包含模拟或占位结果。不得把未逐条核验的检索结果、引文、参考文献或学术主张用于投稿、发表或对外事实陈述。
