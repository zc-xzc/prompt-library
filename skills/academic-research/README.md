# 🎓 Academic Research（学术研究）

科研文献检索、引用管理、论文展示、学术写作工作流。

## 技能

| 技能 | 说明 |
|------|------|
| [bgpt-paper-search](bgpt-paper-search/) | BGPT 科学文献检索，返回 25+ 字段的结构化数据 |
| [nature-academic-search](nature-academic-search/) | 多源学术文献检索（PubMed、CrossRef、arXiv、Semantic Scholar） |
| [nature-citation](nature-citation/) | Nature/CNS 系列学术引文格式添加 |
| [nature-paper2ppt](nature-paper2ppt/) | 科学论文自动转化为中文 PPTX 演示文稿 |

## 提示词

| 提示词 | 说明 |
|--------|------|
| [academic-tools](academic-tools/) | 研究工具选择与使用的安全任务规划 |
| [nature-agent](nature-agent/) | 学术研究全流程工作流：检索 → 写作 → 润色 → 引文管理 |

## 使用

```bash
# 安装技能
cp -r bgpt-paper-search ~/.codex/skills/

# 使用提示词（直接复制到 Codex 对话）
```
