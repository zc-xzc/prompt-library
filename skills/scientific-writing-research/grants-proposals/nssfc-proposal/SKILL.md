# 国家社会科学基金（NSSFC）年度项目申请书撰写

## 概述

本 Skill 指导 AI Agent 协助用户撰写**国家社会科学基金年度项目**申请书，依据 2025 年全国哲学社会科学工作办公室官方模板，覆盖重点项目（A）、一般项目（B）、青年项目（C）、西部项目（X）四类。区别于本仓库已有的 `research-grants`（面向 NSF/NIH 等海外自然科学基金）与 `nsfc-proposal`（面向国家自然科学基金），本 Skill 专为**中国大陆哲学社会科学基金**设计，特别关注：

- **2025 年新增"选题说明"模块**（≤300字，单独成段，评审专家首读内容）
- **活页匿名化要求**（不得出现作者姓名、单位、发表刊物名、合作者等可识别身份信息）
- **学科评审标准差异**（25 个学科分类，各学科评审重点不同）
- **"目标导向、问题意识、学科视角"三大核心评审原则**

**学术诚信声明**：申请书的科学观点、研究思路、创新点必须由申请人本人提供；AI 仅可用于辅助检索文献、整理结构、润色语言，且须人工核实事实准确性。

---

## 何时使用本 Skill

当用户的请求涉及以下情形时应触发本 Skill：

- 撰写、修改或润色国社科申请书的任一模块
- 询问重点 / 一般 / 青年 / 西部项目的申报规范、字数限制、经费额度
- 检查申请书是否符合 2025 年新版要求（选题说明 300 字、单独成段）
- 准备**活页版**（须严格匿名化处理）
- 整理前期成果（限报 5 项，与本课题高度相关）
- 设计经费概算（间接费用≤40%，劳务费一般≤直接费用 30%）

## 四类项目核心差异速查

| 维度 | A 重点项目 | B 一般项目 | C 青年项目 | X 西部项目 |
|---|---|---|---|---|
| 申请人资格 | 副高及以上，或博士学位 | 副高及以上，或博士学位 | 35 周岁以下 | 西部省区在职人员 |
| 资助经费 | 35 万元 | 20 万元 | 20 万元 | 20 万元 |
| 完成时间 | 基础研究3-5年，应用研究2-3年 | 同左 | 同左 | 同左 |
| 评审侧重 | 重大理论与现实问题 | 学科前沿与创新性 | 创新潜力与学术训练 | 西部经济社会发展 |

> 详细规范见 `references/NSSFC_WRITING_SPEC.md`。

## 工作流程

### Phase 1 · 信息收集

通过对话（或在支持的工具中使用结构化提问）收集以下要素：

```
【课题名称】：不超过 40 个汉字，不加副标题
【项目类别】：A 重点 / B 一般 / C 青年 / X 西部
【学科分类】：一级学科 + 二级学科（如：社会学 / 社会学理论）
【研究类型】：A 基础研究 / B 应用研究 / C 综合研究
【关键词】：≤3 个，空格分隔
【申请经费】：A=35 万；B/C/X=20 万
【计划完成时间】：基础研究 3-5 年；应用研究 2-3 年
【预期成果形式】：专著 / 论文集 / 论文 / 研究报告
【申请人背景】：职称、研究方向、代表性成果
【课题组成员】：姓名、职称、单位、研究专长
【核心研究问题】：2-3 句话描述
【前期成果】：与本课题相关的已有成果，最多 5 项
【与已有项目关系】：无 / 有（说明联系与区别）
【是否需要活页版】：是 / 否（决定匿名化处理）
```

如用户信息不全，针对最影响 Agent 决策的字段（**项目类别、学科分类、核心研究问题**）追问；其余可填"请自行补充"占位。

### Phase 2 · 文献检索

参照 `references/LITERATURE_SEARCH.md`，按学科领域选择检索策略：

- **优先**：CNKI（中国知网）、万方、维普 → 中文核心 / CSSCI 来源期刊
- **国际**：JSTOR、SSCI、A&HCI、SAGE、Project MUSE
- **专项数据库**：
  - 法学类：北大法宝、聚法案例、Westlaw、HeinOnline
  - 经济学类：EconLit、NBER、CEIC
  - 历史学类：中华古籍资源库、爱如生数据库、JSTOR
  - 文学类：CNKI 古籍库、中华经典古籍库
- **如果已安装 `qinyan-paper-search` Skill** → 用 `wanfang`（中文）与 `google`（外文）双源

**学科领域取向建议**：

| 学科 | 优先检索库 | 备注 |
|---|---|---|
| 哲学 / 宗教学 / 中国文学 | CNKI → 古籍数据库 → JSTOR | 注重中外文兼顾 |
| 经济学 / 应用经济 | EconLit → CNKI → NBER | 数据来源用 CEIC/Wind |
| 法学 | 北大法宝 → CNKI → HeinOnline | 案例检索用聚法/Westlaw |
| 社会学 / 民族学 | CNKI → 万方 → SSCI | 田野资料须标注获取方式 |
| 政治学 / 国际问题 | CNKI → SSCI → 智库报告 | 用 RAND/Brookings 等智库 |
| 教育学 / 心理学 | CNKI → ERIC → APA PsycInfo | 实证研究优先 SSCI |
| 历史学 / 考古学 | 中华古籍资源库 → CNKI → JSTOR | 史料须辨章学术 |
| 新闻传播 / 图情 | CNKI → Communication Abstracts | 政策类引用《传媒》 |

### Phase 3 · 选题说明 + 选题依据写作

**选题说明（≤300字，2025年新增）**是评审专家首读内容，须高度凝练：

- 一段话连贯表达，**不分条**
- 涵盖三要素：①核心研究问题 ②研究视角 / 理论框架 ③核心概念界定
- 禁出现作者姓名、单位名称

**选题依据（约1500-2000字）**采用 "述：评 = 3:7" 原则：

1. **学术史梳理与研究进展**（约 500 字）：
   - 国际研究：分 2-3 派 / 阶段，代表人物 + 观点 + 局限
   - 国内研究：分 2-3 维度，评述成就与不足
   - 已有国社科同类项目（近 5 年）：指出本研究的差异
2. **学术价值与应用价值**（约 800 字）：
   - 学术价值 2-3 点：填补空白 / 拓展理论 / 提供新框架
   - 应用价值 1-2 点：政策依据 / 决策参考 / 回应国家战略

### Phase 4 · 研究内容 + 创新之处 + 研究基础

**研究内容**（约 1500 字）必须出现**提纲或目录**（框架可视化）：

```
第一部分：导论与研究设计（约 X 千字）
第二部分：【核心议题 A】的理论分析
第三部分：【核心议题 B】的实证检验
第四部分：【核心议题 C】的比较 / 案例分析
第五部分：结论与政策建议
```

**研究计划**按年度展开（表格形式），与"计划完成时间"严格一致。

**创新之处**（约 600 字）必须分条列举，从三维度各选 1-2 点：
- 学术观点创新
- 研究方法创新
- 研究视角 / 议题创新

**研究基础**：
- 学术简历（约 400 字）：与本课题相关的学历、访学、项目经历
- 前期成果（限 5 项）：成果名称 + 形式 + 发表信息 + **与本课题的关联**
- 承担项目情况：表格形式
- 与已有项目 / 博士论文的关系：若有关联须明确"实质性拓展"（文字重复率<60%）

### Phase 5 · 活页版处理与自检

**如果用户需要活页版**：

- 总字数 ≤ 7000 字（含选题说明）
- **严格匿名化**：
  - 删除申请人姓名、单位名称、所在城市（用"某高校""某省"代替）
  - 前期成果删除发表刊物名、出版社名、合作者姓名
  - 删除任何可推断出申请人身份的描述
- 前期成果仅列**成果名称 + 与本课题的关联**，不列具体发表信息

按 `references/QUALITY_CHECKLIST.md` 完成最终自检，输出两份文件：
- `申请书_完整版_YYYY-MM-DD.md`
- `申请书_活页版_YYYY-MM-DD.md`（已匿名化）

## 高频错误警示

| 错误 | 改正 |
|---|---|
| 选题说明分条列点 | 改为一段话连贯表达 |
| 选题依据只"述"不"评" | 每段末加"然而……""但现有研究尚未……"揭示研究缺口 |
| 研究内容无提纲 | 必须出现可视化的部分 / 章节框架 |
| 创新之处空洞套话 | 每条须有实质内容，附文献对比支撑 |
| 前期成果与课题无关 | 仅列与本课题高度相关的成果（≤5 项） |
| 活页版含可识别信息 | 删除姓名、单位、刊物、合作者；用"某高校""某省"代替 |
| 经费概算各科目无说明 | 每个科目须有"用途说明"；劳务费≤直接费用 30% |
| 课题名称带副标题 | 删除副标题，≤40 字反映核心研究问题 |
| 研究计划与完成时间不一致 | 按年度展开，与"计划完成时间"严格对齐 |

> **最终提醒**：申请书须由申请人本人撰写。本 Skill 仅作为结构框架与写作指导，所有学术观点、研究思路必须来自申请人本人的学术积累与学科思考。高质量的申请书体现的是申请人的学科视角与问题意识，而非对模板的套用。
