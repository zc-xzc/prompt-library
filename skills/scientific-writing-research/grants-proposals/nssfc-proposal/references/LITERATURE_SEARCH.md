# 国社科申请书 · 文献检索策略

撰写国家社会科学基金申请书时，文献检索贯穿"选题依据 → 研究内容 → 研究基础"三大模块。社科领域的特点是**中英文文献并重、跨学科融合、政策性文献与学术性文献并存**，因此检索策略与自然科学有显著差异。

---

## 1. 优先使用沁言学术统一检索接口

如果运行环境已安装本仓库的 `qinyan-paper-search` Skill 并配置了 `QINYAN_API_KEY`：

```bash
# 万方（中文社科文献首选）
bash ~/.claude/skills/qinyan-paper-search/scripts/search.sh wanfang \
  '{"query": "你的关键词", "max_results": 50, "date_from": "2020"}'

# Google Scholar（外文综合 + 跨学科）
bash ~/.claude/skills/qinyan-paper-search/scripts/search.sh google \
  '{"query": "your topic", "max_results": 20, "date_from": "2020", "language": "english"}'
```

> 在不同工具中安装目录略有差异：Cursor 用 `~/.cursor/skills/`，Codex 用 `~/.codex/skills/`，依此类推。
>
> **注意**：PubMed 与 ArXiv 主要面向理工医学领域，在社科应用极少；如需使用，仅适用于交叉学科（如经济学量化、计算社会科学、心理学实证研究）。

## 2. 国社科评审权威检索源 · CNKI

国社科评审专家**最常用 CNKI（中国知网）**作为基线参考，因此申请书的中文文献必须能在 CNKI 上找到对应记录。

如果环境未提供 CNKI API（多数情况），建议 Agent：

1. 让用户在浏览器手动检索 CNKI，提供检索关键词组合
2. 用户复制核心 10-20 篇文献的题录至 Markdown，Agent 据此整理综述
3. 必备 CNKI 子库：
   - **学术期刊**：覆盖 CSSCI 来源期刊
   - **中国博士学位论文全文数据库**
   - **国家社会科学基金资助项目成果数据库**（必查，定位"已有同类立项"）

**CNKI 检索式模板**：

```
SU = ("核心研究对象") AND ("核心理论/视角") AND PT = "期刊"
TIME = 2020-2025
SORT BY: 被引用次数 DESC
SOURCE FILTER: CSSCI 来源期刊
```

## 4. 检索式模板

### 4.1 选题依据 — 国际研究综述

```
SU = ("核心概念" English term) AND ("理论视角") AND (review OR meta-analysis OR perspective)
TIME = last 5 years (2020-2025)
SORT BY: citations / journal impact
DATABASES: CNKI 外文 + Web of Science (SSCI/A&HCI) + 学科专业库
```

### 4.2 选题依据 — 国内研究综述

```
SU = ("核心概念" 中文) AND ("理论视角" 中文)
TIME = 2020-2025
FILTER: CSSCI 来源期刊
SORT BY: 被引用次数 DESC
DATABASES: CNKI 学术期刊
```

### 4.3 已有国社科同类立项查询（必做）

CNKI → 知识服务平台 → "项目"
或访问 [国家社科基金项目数据库](http://fz.people.com.cn/skygb/sk/index.php/index/seach)：

```
项目类别：国家社科基金（全部）
关键词：[与本课题核心议题相关]
近 5 年立项
```

**这一步极为关键**：必须找到 3-5 个最相关的近期立项，在申请书中明确说明本课题"与之的差异"（避免重复立项嫌疑）。

### 4.4 研究基础 — 申请人前期成果定位

```
作者 = [申请人姓名]
TIME = 近 5 年
FILTER: CSSCI / SSCI / 核心期刊
```

## 5. 文献综述的写作要点（结合检索结果）

撰写选题依据综述时，Agent 应：

1. **"述：评 = 3:7"**：每段须有自己的评述与判断，不堆砌他人观点。
2. **每段末转折句**："然而……""但现有研究尚未……""值得注意的是……"
3. **国际与国内分别成段**：国际研究先行，国内研究后置，最后比较中外异同。
4. **明确同类项目对照**：单独一段评述近 5 年国社科同类立项，指出本研究的差异。
5. **参考文献格式遵循 GB/T 7714**：
   - 期刊：作者. 题目[J]. 期刊名, 年份(期): 页码.
   - 专著：作者. 书名[M]. 出版地: 出版社, 年份.
   - 英文：Author, A. (Year). Title. *Journal Name*, Volume(Issue), Pages.

## 6. 活页版的检索结果引用注意事项

如需准备**活页版**，引用文献时须额外注意：

- ✅ 允许：作者姓名 + 年份 + 期刊名（无身份指向时）
- ❌ 禁止：将申请人本人的论文以"X 的最新研究"形式引用并附姓名
- ⚠️ **申请人自引**：必须改为匿名表述，如"已有研究（具体研究者姓名匿名）……"

Agent 在协助综述检索后，应特别提醒用户排查活页中的自引隐蔽身份信息。

## 7. AI 辅助检索的合规声明（建议申请人采用）

> 本申请书在文献检索阶段使用了 AI 助手（[工具名/版本]）协助筛选与归类近 5 年中外文献，所有引用的具体观点、数据与结论均由申请人本人核实原文后采纳。申请书正文内容由申请人独立撰写。
