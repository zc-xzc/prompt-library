# 人际交往思考指南：尊重与边界
一份倡导尊重与边界的健康人际交往指南，拒绝套路讨好，回归真诚相处，帮助减少社交内耗。

## ✨ 核心亮点
- 🚫 **10条社交红线自查**：规避过度付出、反复纠缠、深夜情绪化打扰等常见误区
- ✅ **11条可落地正向行动建议**：给出体面、克制的相处方式
- 🤝 **健康友谊判定标准**：清晰区分良性、消耗型关系
- 🧠 **自我反思工具**：8个自查问题，跳出情绪理性复盘人际关系
- 📝 **配套复盘写作框架**：把社交经历转化为个人成长

## 🎯 适用人群
- 习惯性过度付出、讨好他人，长期陷入社交内耗
- 分不清对方礼貌与好感，容易单方面脑补期待
- 被拒绝后不知道如何体面退场，反复纠结关系走向
- 希望建立松弛、平等、互相尊重的健康人际关系

## 📖 快速使用方法
完整提示词：[interpersonal-boundary-guide.md](interpersonal-boundary-guide.md)
1. 先查看「常见误区提醒」，自查自身社交行为是否越界
2. 对照「正向行动建议」调整相处节奏与沟通方式
3. 使用「自我反思问题」梳理情绪，客观判断关系走向
4. 将重心放回自身学习、工作与生活，减少对他人的精神依赖
