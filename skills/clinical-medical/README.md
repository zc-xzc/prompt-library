# 🏥 Clinical & Medical（临床医学）

面向医疗专业人员的临床决策、报告撰写、治疗方案和法规认证技能。

## 技能列表

| 技能 | 说明 |
|------|------|
| [clinical-decision-support](clinical-decision-support/) | 临床决策支持文档 |
| [clinical-reports](clinical-reports/) | 临床报告撰写（病例报告、诊断报告、临床试验报告） |
| [clinicaltrials-database](clinicaltrials-database/) | ClinicalTrials.gov 临床试验数据库 |
| [fda-database](fda-database/) | openFDA 药物与医疗器械数据 |
| [iso-13485-certification](iso-13485-certification/) | ISO 13485 医疗器械质量管理体系认证 |
| [treatment-plans](treatment-plans/) | 治疗方案生成（LaTeX/PDF，SMART 框架） |
